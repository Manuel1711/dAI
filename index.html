About ERC-8004

ERC-8004 introduces a composable on-chain identity layer for AI agents and a feedback registry for verifiable reputation. The goal is to make AI agents discoverable, auditable and interoperable across applications.

Why this observatory

This dashboard provides a live operational view of agent creation, ownership, feedback activity and ranking quality using event-driven indexing from Ethereum L1.

Scoring model (v1)

scoreV1 = arithmetic mean(feedback.score). Revoked feedback is excluded where detected. This gives a simple baseline before robust weighting and anti-spam controls.
